# Raypeak 的数字花园

你好，我是 Raypeak，2008 年出生，一名高二学生。

这个仓库是我在互联网上的一个小角落。我把它叫作“数字花园”，因为这里会慢慢长出一些东西：学习留下的痕迹、看完剧之后的胡思乱想、成长过程中的困惑和顿悟。

如果你偶然路过，欢迎进来坐坐。

## 关于我

**2008年出生 | 高二 | 现居中国**

喜欢《怪奇物语》里那种“普通人在绝境中爆发出勇气”的故事，也喜欢《绝命毒师》那种“一个人如何一步步走向黑暗又试图找回自己”的复杂人性。

现实生活里，和大多数高中生一样，每天和数学题、英语单词打交道。但在做题之外，我喜欢想一些有的没的：比如原生家庭到底在多大程度上决定了我们，比如未来需要什么样的人，比如我到底想成为什么样的人。

这个仓库就是这些思考的容器vessel。

## 这里有什么

我把内容分成了四个文件夹，每个文件夹对应一类记录：

### 📁 学习/

学习的痕迹都在这里。

不是那种“今天学了什么”的流水账，而是真正花时间整理过的东西：

- 数学题的解题思路，不是答案本身，是“我当时怎么想到这一步的”
- 英语课上记下的有用表达，特别是从《绝命毒师》台词里扒下来的
- 复习时发现的规律，比如“这类题型其实都是同一个套路”

文件夹里面再按学科分：数学、英语、物理……以后可能还会增加。

### 📁 影评/

我看剧的方式可能和别人不太一样。

别人看《怪奇物语》看的是剧情，我看的是：

- Max 在面对 Vecna 时，为什么那首歌能给她力量
- Eleven 从小被当作实验品，她怎么学会“成为一个人”
- 这群孩子每次遇到危险，为什么总能选择“不放弃”

《绝命毒师》也是：

- 老白说“I did it for myself”，他真的只是为了自己吗
- Jesse 那么多次可以逃，为什么一次次回来
- 整部剧到底在讲“变坏”，还是在讲“发现自己本来就是这样”

这些思考都会写下来，放在这个文件夹里。

### 📁 想法/

那些突然冒出来的、不属于以上任何一类的念头。

比如有一天突然想：**“极客型”的人在未来会更需要吗？**

比如刷到一篇文章，开始想：**“如果AI什么都能做，那我们学这些还有什么意义？”**

比如看到同学都在卷，开始想：**“我是真的想学，还是只是不想落后？”**

这些想法没有答案，但记录下来本身就有意义。

### 📁 My Philosophical Scrapbook / 哲思剪贴本

这我要详细讲讲

# My Philosophical Scrapbook

一个高二学生的思想碎片 —— 在算法的洪流里，打捞属于人的那部分

## 📖 关于这个章节

这个文件夹里装着的，是我在过去一段时间里，从抖音刷到的那些“有点东西”的视频中摘录下来的文字。起初只是随手记在备忘录里，后来发现攒得多了，就想找个地方存起来——像是给自己建一个数字版的剪贴本。

这里没有高考考点，没有标准答案，只有一些关于**人性、成长、认知**的碎片。它们来自我永远记不住名字的博主，来自偶尔划到的演讲片段，来自那些让我停下来多想了一会儿的瞬间。

说起来有点讽刺——这个文件夹里的内容，都是从**抖音**来的。

就是那个让我一刷就停不下来的App。那个说好只看五分钟、结果两小时过去了的App。那个在我写作业时总想偷偷点开的App。

我知道刷短视频有很多坏处。我知道它在偷我的时间，磨我的耐心，让我越来越难读完一篇长文章。我知道那些15秒的刺激过后，留下的往往是更深的空虚。我知道算法比我更了解我自己——它知道我什么时候该看励志鸡汤，什么时候该看搞笑段子，什么时候该看那些“看完顿悟”的人生哲理。

但它也确实让我刷到了一些东西。一些让我停下来想一想的、让我忍不住截屏记下来的、让我觉得“原来还可以这样看世界”的东西。

这个文件夹，就是我和这个App之间，一场漫长的拉锯战的见证。我一边恨它，一边离不开它；一边被它消耗，一边从它那里打捞点什么。

说实话，我也不知道这样做的意义是什么。

可能只是因为，刷到有些内容的时候，心里会“咯噔”一下——那种感觉就像：*“原来还可以这样想问题？”* 或者 *“这句话好像戳中了我一直说不出来的东西。”*

我怕这些瞬间刷过去就忘了，所以把它们留了下来。

某种意义上，这个剪贴本是我对算法的**微小反抗**——把那些本该划走的瞬间，变成可以反复看、慢慢想、真正消化掉的东西。

整理下来发现，我摘录的东西好像总是绕着这几个话题打转：

- **关于“见过世面”**——不止是去过多少地方，更是能看懂多少种活法
- **关于动物们的隐喻**——狮子怎么工作，鳄鱼怎么等待，章鱼怎么长出触手
- **关于做个清醒的人**——在人群里保留一点孤独，在热闹中留一只眼睛看自己
- **关于那些说不清的感受**——“你在干嘛”背后的想念，废墟上飞过的蝴蝶

内容很杂，有长有短，有的像鸡汤，有的像哲学课。但都是我现阶段觉得“有用”或“美”的东西。

这些文字是**碎片**，不是答案。

短视频的问题之一，就是把复杂的世界压缩成几分钟的“顿悟”。好像听完一个道理，人生就能变好。我不信这个。

所以，如果你读到什么有共鸣的句子，别停下来。顺着它，去找原书、找原文、找更完整的思考。我的剪贴本只是一扇门，门后面还有很长的路。

我知道很多想法以后可能会变。

也许过几年回头看，会觉得这些文字幼稚、矫情、故作深沉。

但那也挺好的——至少能看见自己是怎么一路长过来的。

从一个被算法喂养的少年，慢慢长成一个会自己找东西吃的人。

如果你偶然刷到这个仓库，并且愿意读几页：

愿你有被看见的幸运，也有不去干涉的克制。

愿你的废墟上，终将开出自己的花。


*“笼子没有锁。门，就在你每一次想都不想就要滑动的时候，关上的。”*

## 为什么做这件事

有三个原因。

**第一，给自己一个交代。**

每天刷题、考试、排名，很容易忘记“我到底在干什么”。写下来，是为了提醒自己：我在长大，我在思考，我不只是一个学生编号。

**第二，给未来留一份礼物。**

想象一下：五年后、十年后的我，回来看这个仓库，看到17岁时写下的这些文字——那些困惑、那些幼稚的想法、那些突然的顿悟——一定会很有意思。这是给未来的自己准备的时光胶囊。

**第三，给同类一个信号。**

我知道在这个年纪，像我这样想这些乱七八糟问题的人不多。但一定存在。如果你看到了这里，觉得“这人有点意思”，那我们就可以在 Issues 里聊聊天。不一定聊什么，随便什么都行。

## 一些规则

这个仓库虽然公开，但它首先是给我自己的。所以：

- 内容很随意，想到什么写什么，不保证质量
- 更新时间不固定，可能每天，可能一周一次
- 观点随时可能变，今天的我和明天的我可能吵架

但如果你愿意来：

- 欢迎在 Issues 里说话，讨论剧、讨论学习、讨论人生，都行
- 如果发现我写错了什么，告诉我，我会很开心
- 如果想要 Star，我当然更开心，那是给我的鼓励
