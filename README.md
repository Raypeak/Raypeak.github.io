## 📖 关于这个章节

这个文件夹里装着的，是我在过去一段时间里，从抖音刷到的那些“有点东西”的视频中摘录下来的文字（持续更新中）

说起来有点讽刺——这个文件夹里的内容，都是从**抖音**来的

我知道刷短视频有很多坏处。我知道它在偷我的时间，磨我的耐心，让我越来越难读完一篇长文章。我知道那些15秒的刺激过后，留下的往往是更深的空虚。我知道算法比我更了解我自己——它知道我什么时候该看励志鸡汤，什么时候该看搞笑段子，什么时候该看那些“看完顿悟”的人生哲理。

但它也确实让我刷到了一些东西。一些让我停下来想一想的、让我忍不住截屏记下来的、让我觉得“原来还可以这样看世界”的东西。

这个文件夹，就是我和这个App之间，一场漫长的拉锯战的见证。我一边恨它，一边离不开它；一边被它消耗，一边从它那里打捞点什么。

说实话，我也不知道这样做的意义是什么。

可能只是因为，刷到有些内容的时候，心里会“咯噔”一下——那种感觉就像：*“原来还可以这样想问题？”* 或者 *“这句话好像戳中了我一直说不出来的东西。”*

我怕这些瞬间刷过去就忘了，所以把它们留了下来。

某种意义上，这个剪贴本是我对算法的**微小反抗**——把那些本该划走的瞬间，变成可以反复看、慢慢想、真正消化掉的东西。

内容很杂，有长有短，有的像鸡汤，有的像哲学课。但都是我现阶段觉得“有用”或“美”的东西。


短视频的问题之一，就是把复杂的世界压缩成几分钟的“顿悟”。

我的剪贴本只是一扇门，门后面还有很长的路。

我知道很多想法以后可能会变。

也许过几年回头看，会觉得这些文字幼稚

但那也挺好的——至少能看见自己是怎么一路长过来的。


如果你偶然刷到这个仓库，并且愿意读几页：
